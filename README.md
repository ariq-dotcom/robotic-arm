Repository ini berisi file-file pengembangan robot arm 5 DOF berbasis Arduino Mega, meliputi desain elektrikal (PCB), program kendali

Pada folder ARM KENDALI, setiap angka pada nama file .ino menunjukkan urutan perkembangan dan pembaruan program sejak tahap awal pengembangan, ketika isi program masih berupa struktur dasar dan belum sepenuhnya siap digunakan. File dengan angka terbesar merupakan versi program terbaru.
